# 🕌 أدوات الصوت الديني والمشيخي

## أدوات خاصة بالمحتوى القرآني والديني

### 1. Tarteel AI
- الرابط: https://github.com/tarteel-ai
- النوع: تتبع تلاوة القرآن بالذكاء الاصطناعي في الوقت الفعلي
- مناسب لـ: تعلم التلاوة، التحقق من صحة التلاوة

### 2. Quran MD Dataset
- الرابط: https://github.com/umar1997/quran-md
- النوع: مجموعة بيانات صوتية للقرآن الكريم (187,000+ ملف صوتي)
- مناسب لـ: تدريب نماذج TTS بصوت قرآن حقيقي
- يتضمن: أصوات 30 قارئ مختلف

### 3. Quran Recitation Audio Classification
- الرابط: https://github.com/usmana0820/Quran-Recitation-Audio-Classification
- النوع: تصنيف أساليب التلاوة بالتعلم الآلي

### 4. Awesome Arabic AI
- الرابط: https://github.com/OmarSalah26/Awesome-Arabic-AI
- النوع: مرجع شامل لنماذج الذكاء الاصطناعي العربي
- يتضمن: TTS عربي، STT، نماذج لغوية

### 5. ArTST - Arabic TTS Transformer
- الرابط: https://github.com/mbzuai-nlp/ArTST
- النوع: نموذج Transformer مخصص للغة العربية
- أفضل من Whisper للعربية الفصحى

## 🎯 كيفية تدريب صوت شيخ / مشيخ

### الخطوات:
1. جمع عينات صوتية للشيخ (دقيقة واحدة على الأقل)
2. تنظيف الصوت: إزالة الضجيج، تسوية المستوى
3. استخدام XTTS-v2 أو F5-TTS للاستنساخ
4. إضافة مؤثرات صوتية تناسب المحتوى الديني
5. معالجة نهائية بالإيكو والريفرب الخفيف
