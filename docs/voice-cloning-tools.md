# Voice Cloning Tools

هذه القائمة تركز فقط على الأدوات مفتوحة المصدر التي تدعم استنساخ الصوت أو تحويله:

- Coqui TTS / XTTS-v2 — https://github.com/coqui-ai/TTS
- F5-TTS — https://github.com/SWivid/F5-TTS
- OpenVoice — https://github.com/myshell-ai/OpenVoice
- Fish Speech — https://github.com/fishaudio/fish-speech
- RVC — https://github.com/RVC-Project/Retrieval-based-Voice-Conversion-WebUI

## الترتيب المقترح للجودة

1. F5-TTS
2. XTTS-v2
3. OpenVoice
4. Fish Speech
5. RVC

## ملاحظات

- أفضل نتيجة تأتي من عينة صوت نظيفة.
- جودة العربية تختلف حسب النموذج.
- لا يوجد ضمان مطلق لصوت 100% بشري، لكن يمكن الوصول إلى جودة عالية جداً.
